# My Website - [ngn13.fun](https://ngn13.fun)

This repo contains the source code of my personal website.
It's written NuxtJS and supports full SSR. As database,
it uses mongodb.

## Setup

```
git clone https://github.com/ngn13/ngn13.fun.git
cd ngn13.fun && npm i
echo "password" > .env
npm run build
npm run start
```
